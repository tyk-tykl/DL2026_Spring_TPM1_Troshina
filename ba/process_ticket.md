# Процесс покупки билета

1. Вход на сайт
 Пользователь открывает сайт и переходит к сервису покупки билетов.
2. Ввод параметров поездки
 Пользователь вводит станцию отправления, станцию назначения и дату поездки.
3. Поиск поездов
 Система выполняет поиск поездов, соответствующих заданным параметрам.
4. Проверка наличия поездов
 Если по заданному маршруту поезда отсутствуют, пользователю предлагается изменить параметры поиска. Если поезда найдены - отображается список доступных вариантов.
5. Просмотр доступных поездов
 Пользователь просматривает предложенные варианты, включая время отправления, прибытия и стоимость билетов.
6. Выбор поезда и места
 Пользователь выбирает подходящий поезд, тип вагона и свободное место.
7. Ввод данных пассажира
 Пользователь вводит персональные данные пассажира (ФИО, пол, дата рождения, тип и номер документа, гражданство и номер телефона).
8. Проверка корректности данных
 Система проверяет правильность введённой информации. При обнаружении ошибок пользователю предлагается исправить данные.
9. Подтверждение покупки
 Пользователь подтверждает намерение приобрести билет. В случае отказа процесс покупки прекращается.
10. Переход к оплате
 Пользователь переходит на страницу оплаты.
11. Выбор способа оплаты
 Пользователь выбирает доступный способ оплаты (наличный/безналичный расчёт).
## Обработка возможных ошибок
- сбой соединения
- нет поездов
- ошибка оплаты
- отмена
- другие ошибки

## Завершение
- Подтверждение оплаты
 После успешной обработки платежа система подтверждает оплату.
- Формирование билета
 Система автоматически формирует электронный билет.
- Отправка билета пользователю
 Электронный билет отправляется на электронную почту пользователя и отображается на сайте.
